Gestión de Usuarios

En esta sección puedes gestionar la información de los usuarios.

Puedes buscar un usuario solo por ID usando la barra de búsqueda.

📌 Funciones disponibles:

  • 🔍 Buscar Usuario: Introduce el ID y encuentra su información.
  • ➕ Añadir Usuario: Haz clic en "Añadir" y aparecerá una ventana para registrar uno nuevo. No se puede repetir el nombre de usuario, las contaseñas deben coincidir y se puede marcar la casilla de es admin o no para que tenga los persmisos admin.
  • ✏ Editar Usuario: Selecciona un usuario de la lista y haz clic en "Editar" para modificar su información.
  • 🗑 Eliminar Usuario: Selecciona un usuario y haz clic en "Eliminar" para quitarla del sistema.
