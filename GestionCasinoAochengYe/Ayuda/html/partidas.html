Gestión de Partidas

En esta sección puedes gestionar la información de las partidas en el casino.

Puedes buscar una partida solo por ID usando la barra de búsqueda.

📌 Funciones disponibles:

  • 🔍 Buscar Partida: Introduce el ID y encuentra su información.
  • ➕ Añadir Partida: Haz clic en "Añadir" y aparecerá una ventana para registrar una nueva.
  • ✏ Editar Partida: Selecciona una partida de la lista y haz clic en "Editar" para modificar su información.
  • 🗑 Eliminar Partida: Selecciona una partida y haz clic en "Eliminar" para quitarla del sistema.
